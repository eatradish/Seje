# saki.li
A boring Hexo theme

## Use
- Edit _config.yml
- `yarn add hexo-excerpt`
- `hexo g -d`

### If you use RSS
- `yarn add hexo-generator-feed`
- `edit $BLOG/themes/saki.li/_config.yml` and add `disqus-shortname: `
- `hexo g -d`

### If you use disqus
- Set your disqus_shortname in your hexo `_config.yml`
- `hexo g -d`
